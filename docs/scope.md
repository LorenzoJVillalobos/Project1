# Purpose

To create a personal introduction webpage for CMPA 3301 that demonstrates foundational HTML structure and introduces who I am, my academic goals, and professional interests.

## Description

This project consists of a simple HTML webpage about me. The page will include my name, educational background, career goals, personal interests, and a link to my GitHub profile. The webpage will be organized using semantic HTML elements and will focus on proper structure rather than visual design.

## Desired Results

1. Create a valid index.html file using required HTML tags.
2. Include my name, educational and career information, and personal interests.
3. Include a working hyperlink to my GitHub profile.
4. Use proper semantic HTML structure.
5. Submit both scope.md and index.html in a single ZIP file before the deadline.

## Exclusions

- No CSS styling.
- No JavaScript.
- No images.
- No external frameworks or libraries.
- No advanced web design features.
