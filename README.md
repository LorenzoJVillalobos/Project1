# Project 03 Final Project – Personal Portfolio Website

## Overview

This repository contains my final project for CMPA 3304. The purpose of this project is to create a professional portfolio website that showcases my background, education, work experience, technical interests, and coursework completed during the semester.

The website was developed using HTML and CSS and is published through GitHub Pages.
